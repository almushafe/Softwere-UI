# PySide6 UI Template - ChatGPT Style Interface

This is a simple desktop GUI application built using **PySide6 (Qt for Python)**. It mimics a sleek modern layout inspired by ChatGPT’s UI. The interface includes a header with login/signup buttons, a centered title, and a stylish text input panel with buttons like Attach, Search, Study, and Voice.

---

## 📸 UI Features

- Custom top header with:
  - Logo icon
  - "Log in" and "Sign up for free" buttons
- Center section with:
  - Title label
  - Rounded input box for user text
  - Soft shadow effects
  - Utility buttons (Attach, Search, Study, Voice)
